Parking Kitty

    Key Words

  • Pay to Park
  • Simple
  • Portland
  • Easy

Parking Kitty has been an app I've wanted to design since I first started using it. The app features a convenient way to pay for parking in Downtown Portland instead of using a pesky meter. The initial app had a lot of similar information on each page which made the “convenient” process to be more confusing than helpful. This app is mainly used by locals, however anyone can sign up and use Parking Kitty!

The user flow I focused on was around the Pay to Park section of the app. Creating a homepage helped with the inclusion of a nav bar that features three sections: Home, Park, and Settings. Home allows users to input their location and find nearby zones. This gives the user crucial information on parking and zone details. After the user has found the closest zone, they can start the 5 step process of paying to park.
